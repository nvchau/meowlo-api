# NodeJS + ExpessJS + MongoDB - Build an app like Trello (API)

## Requirement
* **nodejs >= v14.7.0**
* **npm >= v6.4.7**
* **yarn >= v1.19.1**

Clone project and run test.

```
$ git clone https://github.com/nvchau/meowlo
$ cd meowlo
$ yarn | npm install
$ yarn | npm start
```
